## Setup Instructions

1. **Install Flow CLI**
   ```bash
   sh -ci "$(curl -fsSL https://storage.googleapis.com/flow-cli/install.sh)"
   ```

2. **Navigate to Project**
   ```bash
   cd WalletManager
   ```

3. **Install Dependencies**
   ```bash
   flow dependencies install
   ```

4. **Start Flow Emulator (for testing)**
   ```bash
   flow emulator start
   ```

5. **Deploy Contract**
   ```bash
   flow project deploy
   ```
